# Vision → Plan → Action / 비전 → 계획 → 실행

- English name: Vision → Plan → Action
- Korean name: 비전 → 계획 → 실행
- Selection category: planning
- Use when: Use when a future vision should be made credible through plan and action.
- Avoid when: Avoid when the vision is not yet defined.

## Writing Pattern

**Use this fixed flow**

- Present the Vision
- Strategies and plans to realize it
- Action plans and execution scenarios
- Connect the steps with clear logic—Vision → Strategy → Execution.

**Each major section should include**

- Section heading
- Core paragraph purpose (1–2 sentences)
- Emphasize concreteness and realism so the reader feels, “We can realize this vision together.”

## Example Writing Outline

1. **A Vision for the Future**: By 2030, we will become a global leader built on sustainable technology. (Vision)
2. **Core Strategic Directions**: We will realize the vision via three pillars: innovation, market expansion, and talent. (Plan)
3. **Roadmap and Work Plan**: A three‑phase execution roadmap lays out the annual milestones. (Plan)
4. **Execution Scenario A: Domestic Expansion**: Complete a tailored services program for local customers by 2026. (Action)
5. **Execution Scenario B: Global Entry**: From 2027, expand into the U.S. and Southeast Asia. (Action)
6. **Conclusion and Commitment**: This vision is a survival strategy—not optional—and execution starts now. (Action)

## Writing Notes

- Vision must be a concrete, relatable future state—not a vague declaration.
- Plans must consider feasibility and resources.
- In Action, include phased scenarios or priorities; measurable KPIs strengthen credibility.
- Use an inspiring, motivating tone so the reader feels hope and resolve.
