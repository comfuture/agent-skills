# Quote → Interpretation → Application / 인용 → 해석 → 적용

- English name: Quote → Interpretation → Application
- Korean name: 인용 → 해석 → 적용
- Selection category: reflection
- Use when: Use when a quote, sentence, or external voice should become the interpretive anchor.
- Avoid when: Avoid when the quote is decorative rather than central.

## Writing Pattern

**Use this three-step sequence**

1. Present a quote
2. Interpret the quote
3. Propose applications based on the quote

**Each major section should include**

- Section heading
- Core paragraph purpose (1–2 sentences)
- Link the quote to real context logically, and present applications in actionable form.

## Example Writing Outline

1. **The Quote**: “Direction matters more than speed.” — Mahatma Gandhi
2. **Interpreting the Quote**: Moving quickly in the wrong direction wastes effort.
3. **How to Apply It**: Reaffirm our direction now and revise our execution plan accordingly.

## Writing Notes

- Choose meaningful quotes—famous figures, proverbs, or lines from books.
- Interpretation should adapt the quote’s core value to our situation and include insight beyond summary.
- Application should connect to the current topic or project with specific actions.
- This structure suits brand philosophy, leadership messages, culture change, and value-led strategy talks.
