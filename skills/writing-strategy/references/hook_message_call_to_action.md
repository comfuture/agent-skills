# Hook → Message → Call-to-Action / 주목 → 메시지 → 행동 요청

- English name: Hook → Message → Call-to-Action
- Korean name: 주목 → 메시지 → 행동 요청
- Selection category: persuasion
- Use when: Use for short persuasive writing where attention and action matter more than deep explanation.
- Avoid when: Avoid for long-form analytical drafts.

## Writing Pattern

**Use a fixed three-part structure**

- Hook
  - Use a striking question, statistic, quote, or problem to grab attention.
  - Engage either emotionally or logically.
- Message
  - Summarize the core point persuasively.
  - Clarify why it matters and what it means for the reader.
- Call-to-Action
  - Ask for a concrete action the reader can take now.
  - Clarity and direction matter (e.g., “Buy now,” “Sign up,” “Join us”).

## Example Writing Outline

1. **How Many Hours Do You Lose Daily?**: Office workers average only 2.5 hours of deep focus per day. (Hook)
2. **Our Time Is Disappearing**: Unnecessary meetings and prep work are dragging productivity down year after year. (Message)
3. **Join the Change Now**: Subscribe to our document automation solution and reclaim an hour every day. (Call-to-Action)

## Writing Notes

- For the Hook, trigger an immediate reaction—emotion, curiosity, shock, or empathy.
  - Example: “We write 12,000 reports a year.”
- For the Message, deliver a single core point persuasively; avoid long-winded explanations.
- For the CTA, don’t end with “thank you”—direct the next step explicitly.
  - Example: “Scan the QR code now,” “Book a consultation,” “Propose this solution.”
- Keep transitions natural so the emotional curve rises toward action.
